EduTrack — Frontend (Vite + React)

Setup:
1. Create a copy of .env.example as .env and set VITE_API_BASE if needed.
2. Install:
   npm install
3. Start dev server:
   npm run dev
4. Open: http://localhost:5173

Notes:
- Frontend expects backend endpoints (see services/authService.js & studentService.js).
- For now demo fallback data is present when backend is not ready.
